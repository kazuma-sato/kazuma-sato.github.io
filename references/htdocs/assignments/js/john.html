JavaScript DOM Animation

1. Add an image that you want to move and give it an id.
2. Make an button with an onclick that calls a function.
3. Make a variable that equals document.getElementById("YOUR IMGAGES ID")
3. Then make 2 functions, one for initialization, another for movement.
4. In the initialization function add 2 lines
duck.style.position='relative';
duck.style.left='0px';
5. In the movement function add duck.style.left = parseInt(duck.style.left) + 10 + 'px';
6. Finaly add window.onload = inititialization function();

Useful when you want to make text or images move around the page.
Works with onclick, onblur, onmouseover, onmouseout.

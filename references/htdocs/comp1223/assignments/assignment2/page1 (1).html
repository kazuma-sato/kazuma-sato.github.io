Audio embedding with HTML5 and non-HTML5

by adrian hrendus

This page will teach you how to embed an audio file into a web page using both HTML5 and non HTML5 methods.

Audio embedding is used when you want to provide a visitor with a recording without him/her having to leave your page to listen to it.

Tutorial

  1. Use this code to embed an audio file to your page.
    <audio controls="controls">
       <source src="yourURL.mp3">
    </audio>
  2. If you want the audio to start playing as soon as the page loads add autoplay="" to your code.
    <audio controls autoplay="">
       <source src="yourURL.mp3">
    </audio>
  3. The audio player can also be styled.
    <audio id="player" controls autoplay="">
       <source src="yourURL.mp3">
    </audio>
    Now use CSS to change style.

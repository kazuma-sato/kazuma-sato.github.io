Google Fonts

by harsimran singh bagri

Google Fonts is an interactive directory of free hosted application programming interfaces for web fonts and it helps to change the style and size of characters. You can use Google Fonts to make our page more beautiful, readable and accessible. Moreover, it provides high-quality fonts that you can include in your web pages using the Google Fonts API.

Check out this list of Google Fonts.

Tutorial

  1. Add a special style link to your html file.
    like:

    <link rel="stylesheet" type="text/css" href="http://fonts.googleapis.com/css?family=Tangerine">

  2. Choose the tag by selector accordingly and edit the fonts inside the selector. like:

    p {
    font-family: 'Indie Flower', cursive;
    font-size: 48px;
    }

  3. Copy it to CSS style and name the tag we want.

Examples

p {
font-family: 'Tangerine', serif;
font-size: 48px;
}

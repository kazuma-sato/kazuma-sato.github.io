Tags and Attributes

  1. <a>
    1. href
      • specifies the url of the page the link goes to
    2. target
      • specifies where to open the linked page
  2. <br>
  3. <code>
  4. <img>
    1. alt
      • gives alternate text for an image
    2. height
      • specifies the height of the image in the page in pixels
    3. src
      • specifies the source of the image
    4. width
      • specifies the width of the image in the page in pixels
  5. <li>
    1. type
      • specifies the type of bullet point to use for this specific list item
    2. value
      • allow the change the ordering value of this list item. allows change in bullet value to go out of sequence; 1, 2, 4, 3,
  6. <meta>
    1. charset
      • specifies the character encoding for the html document
    2. content
      • gives the value associated with the http-equiv or name attribute
    3. name
      • specifies a name for the metadata
  7. <ol>
    1. reversed
      • specifies that the list order should be descending
    2. start
      • specifies the start value of an ordered list
    3. type
      • specifies the kind of marker to use in the list

Document Object Model Manipulation with CSS Stying

by kazuma sato

HTML documents rely heavliy on style for a more user-friendly and appealing experience for the user to format and display content off a browser. In web development, Cascading Style Sheets(CSS) is the style sheet language used. There are multiple ways for CSS to be applied to a page: Internal CSS, external CSS and indivisial stying with the style attribute of each element.

Just like in the previous tutorials, JavaScript can easily access <style> tag in a page or even the style attribute of any element in the document. The following is a table of the properties of the Style Object.

Tutorial

Previously we used the style object to change the background colour of the rows on a table. In this tutorial, will be changing the entire page.

  1. First, let's change all the fonts: var elements = document.querySelectorAll( 'body *' );

        for (i in elements){
            elements[i].style.fontFamily = '"Comic Sans MS", cursive, sans-serif';
        }
  2. Now, let's change some colours: var textElements = document.querySelectorAll("a, p, h1, h2, code, td, li, th");
    var darkerBackgrounds = document.querySelectorAll("table tr:nth-child(even), code.codeblock");

    for (i = 0; i < textElements.length; i++)
        textElements[i].style.color ="#3C3D3A";
    for (i = 0; i < darkerBackgrounds.length; i++){
        darkerBackgrounds[i].style.backgroundColor = "white";
        darkerBackgrounds[i].style.color = "#707070";
    }
    document.getElementsByTagName("body")[0].style.backgroundColor = "#DDD4BC";
    document.getElementsByTagName("header")[0].style.background = "linear-gradient(#FBA827, #F8DC0B)";
    document.getElementsByTagName("footer")[0].style.background = "linear-gradient(#F8DC0B,#FBA827)";
  3. Now Let's try moving things around: var p = document.querySelectorAll("p");

    for (var i = 0; i < p.length; i++) {
        p[i].style.width = "50%"
    }
    document.getElementsByTagName("body")[0].style.width = "100%";
    document.getElementsByTagName("body")[0].style.margin = "0";
    document.getElementsByTagName("body")[0].style.maxWidth = "100%";
    document.getElementsByTagName("table")[0].style.width = "50%";
